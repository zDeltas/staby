Information

This help document offers insight and better explanations for Silky Shark's settings and features.

Note from the author

Silky Shark was created as another solution to enable mouse smoothing or stabilization for applications and web canvases that do not natively support smoothing or stabilization.

Being mostly a personal project at the time, this software operates on the notion of "it works fine for me". It will probably help if you have .NET updated, Windows 10, a Wacom or similar tablet that has the option to disable Windows/Microsoft Ink, and have a relatively decent rig. There is no guarantee this will work for everyone given the nature of performance, hardware, and software discrepancies across systems. For a poor guesstimation on hardware and system compatibility, fixes and suggestions to what may be common issues, refer to the Troubleshooting section.

Unfortunately I offer little to no technical support for this software aside from what you will find in this help document. If you need a more personalized solution you are welcome to modify the source code. Silky Shark is free to use and open source software.

Main Window

The Main Window is the first window visible when the application starts. If Silky Shark.config exists in the same directory, Silky Shark will attempt to load settings from it. If not, the program will load with default settings.

Smoothing On Off

Turns the smoothing feature on or off.

Cycle Display

Cycles through the available screens and displays the overlay on them. Useful for when the main drawing screen isn't the primary screen.

Eyedropper

Changes the virtual cursor's main color. To change the fill color refer to the cursor tab in the Settings window.

Strength

Decreases or increases the smoothing strength. This works by changing the time between sampling the virtual cursor's position. Increasing this setting will smooth movement more at the expense of accuracy.

Interpolation

Decreases or increases the number of points between strength sampling. This works by calculating a smooth curve between strength sample points and inserting (interpolating) between them. Increasing this setting will offer cleaner lines at the expense of processing power.

Manual Interpolation

By default the interpolation setting will be automatically calculated as strength decreases or increases. Checking this box overrides this behaviour and allows the manual setting of the interpolation slider and text input. Useful if you need to decrease interpolation for better performance.

Stay On Top

Toggles if the Main Window should stay on top of every window.

Smooth On Draw

Attempts to smooth only when the mouse or pen tablet is down.

Tablet Mode

Indicates if mouse or tablet pen movement is detected and changes the way Silky Shark handles that input. If you're having issues with tablet pen detection, tablet mode can be manually set if Disable Auto Detection is enabled in the tablet tab in the Settings window. Manually toggling tablet mode can only be done when smoothing is off unless done with it's hotkey.

Settings

The Settings window has advanced settings for handling the overlay, cursor, and tablet. Several global hotkeys can also be set here. While the Settings window is open Save Config, Restore Defaults, and triggering hotkeys are disabled.

Disable Overlay

Disables or enables the virtual cursor's overlay. Disabling the overlay will increase performance.

All Screens (Experimental)

Attempts to extend the virtual cursor's overlay to all screens if applicable. This is very performance heavy.

Manual Overlay Override

Enables the overlay override text inputs and overrides the location and size of the overlay. Overriding the overlay to occupy a smaller size will increase performance.

Overlay Size

Sets the overlay's width in terms of x pixels and height in terms of y pixels.

Overlay Position

Sets the overlay's position in terms of x and y in pixels.

Disable Catch Up

Disables the real cursor from catching up to the virtual cursor. Useful if you're having difficulties with lost mouse input when it's not moving.

Snap To Cursor (Smooth On Draw)

By default when Smooth To Draw is enabled the cursor will attempt to snap to the virutal cursor's position when done drawing. This will disable that behavior and snap the virtual cursor to the real cursor instead. This feature will most likely not work for tablet pens.

Cursor Graphic

Choose from six different graphics to represent the virtual cursor.

Main Color

Mimics the eyedropper functionality and changes the virtual cursor's main color.

Fill Color

Changes the virtual cursor's filling color when applicable.

Tablet

Advanced tablet settings.

Disable Auto Detection
Tolerance
Tablet Offset Override

Disable Auto Detection

This disables automatic tablet detection and enables the option to manually toggle tablet mode in the Main Window. Tabet mode can only be set when smoothing is not on unless done through tablet mode's hotkey. By default Silky Shark will attempt to do automatically detect when a tablet pen is in use by checking raw coordinate input. When a mouse moves it returns a pixel difference representing the next cursor postion whereas a tablet pen will return an absolute position. The absolute position is almost always a large number between 0 and 65536.

Tolerance

Tolerance polls a coordinate to test from the upper left corner of the screen space. Too large of a tolerance will deny tablet usage of the top left corner of the screen space. Too low of a tolerance may lead to fast mouse movements as being misinterpreted as tablet pen movement.

Tablet Offset Override

Overrides the virtual cursor and real cursor's position in terms of x and y in pixels when enabled.

Hotkeys

Silky Shark can set global hotkeys. Note that these may override other application defined hotkeys if they share the same hotkey. Hotkeys are disabled while the Settings window is open.

Hotkeys can be set for the following functions:

• Smooth On/Off - Toggles smoothing on or off.
• Overlay On/ - Toggles the overlay on or off.
• Toggle Display - Cycles through the available monitors to display the overlay on.
• Tablet Mode - Manually toggles Tablet Mode when Disable Auto Detection is off.
• Strength Up - Increases the smoothing strength by 10 in increments of 10.
• Strength Down - Decreases the smoothing strength by 10 in increments of 10.

Tablet cursor jumps between itself and virtual cursor

You will need to disable Windows/Microsoft Ink in your tablet driver's settings. Most if not all Wacom tablets should have this option as a checkbox when setting up program profiles.

How to get this to work on Photoshop

1. Disable Windows/Microsoft Ink for Photoshop in your tablet driver/profiler
2. Navigate to your Photoshop settings folder. It should be located at:
C:\Users\YOURUSERNAME\AppData\Roaming\ Adobe\Adobe Photoshop CC\Adobe Photoshop CC Settings\
3. Locate the settings file: PSUserConfig.txt
4. If it does not exist, create it.
5. Add the following line inside that file:
UseSystemStylus 0

This will enable Photoshop's legacy handling for tablets.

Mouse doesn't finish drawing while left click is down

Under Settings -> Cursor check Disable Catch Up.

Tablet pen is drawing elsewhere

Under Settings -> Tablet check Tablet Offset Override. You can offset the virtual cursor and real cursor with x and y in terms of pixels.

How to improve performance

Disabling the overlay will boost performace by a lot. If you absolutely need the overlay, you can override the overlay with the Manual Overlay Override setting and tweak it to be positioned in an exact area at a smaller size. Enabling Smooth On Draw so smoothing will only occur when drawing may also help.

How to load a configuration file

On startup, Silky Shark will load Silky Shark.config if it's found in the same directory.

Configuration file won't load

Silky Shark.config might be corrupted or may have a formatting error. Delete it and generate a new config file with Save Config.
